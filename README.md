# Aswin Anil Bindu – Portfolio

🌐 **Live site:** [aswinab97.github.io/aswin-portfolio](https://aswinab97.github.io/aswin-portfolio/)

A clean, responsive personal portfolio for a Data Analyst specialising in **Healthcare Analytics** and **Applied AI**, built with vanilla HTML & CSS and deployed via GitHub Pages.

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 (semantic) |
| Styling | CSS3 (custom, responsive) |
| Animations | [AOS](https://michalsnik.github.io/aos/) |
| Icons | [Font Awesome 6](https://fontawesome.com/) |
| Font | [Inter – Google Fonts](https://fonts.google.com/specimen/Inter) |
| Hosting | GitHub Pages |

---

## ✨ Features

- ⚡ Single-page layout with smooth scroll navigation
- 📱 Fully responsive – mobile hamburger menu included
- ♿ Accessible – skip link, ARIA labels, semantic HTML
- 🔍 SEO-ready – meta description, Open Graph tags
- 🎞 Subtle AOS scroll animations
- 📄 Downloadable resume button

## 🚀 Running Locally

No build tools required — it's plain HTML/CSS.

```bash
git clone https://github.com/Aswinab97/aswin-portfolio.git
cd aswin-portfolio
open index.html
```
