Brain MRI Tumor Detector

Python · Deep Learning · Docker · Azure · Flask

Built a Dockerized Python web application that uses a deep learning model to detect brain tumors from MRI images. Users upload a brain MRI scan and the app classifies it as "tumor" or "no tumor" with confidence scoring.

  • Trained a CNN-based deep learning model on labeled MRI image datasets.
  • Containerized the app using Docker and deployed via Azure Container Registry and Azure App Service.
  • Built an end-to-end pipeline from image upload to real-time prediction output.
